# 小土的数据工具箱

## 依赖:

### 1. gtest
* git clone https://github.com/google/googletest.git
* cd gooletest
* mkdir build
* cd build
* cmake ..
* make
* sudo make install

## 使用方式:
* git clone https://github.com/gaoyichao/XiaoTuDataBox.git
* cd XiaoTuDataBox
* make
* ./build/t_${target}.exe
